# Psy Civilization · Psy 文明

A civilization-scale exploration of Psy Protocol as a candidate substrate for the next layer of human and machine coordination — placed in the historical sequence of writing, coinage, double-entry bookkeeping, the joint-stock corporation, the telegraph, the internet, and generative AI. Bilingual (EN / 中文), dark/light, cinematic civilization-OS aesthetic with Syne + Geist Sans + Geist Mono on deep ink-black with electric purple, neon blue, glowing cyan, gold, and rose accents. Restrained motion (one animated flow line, no neon spam), Apple/Stripe-style polish, large hero typography with gradient inflection only on a single phrase. Ten modules: a six-layer architecture diagram (apps → identity → state/realms → execution → coordinator + recursive ZK → L1 settlement) with halos and labelled vertical flow + four detail cards (realms as civilizational primitive, recursive validity, coordinator, persistent identity); civilization infrastructure timeline (writing 3200 BCE → coinage → printing → double-entry → joint-stock → telegraph → ARPANET → web → Bitcoin → generative AI → Psy Protocol) rendered as a horizontally-scrolling node graph; six-row shared-state comparison table (institutional vs protocol-native solutions to ownership, contract execution, attribution, replication, AI capability, accountability); AI-native infrastructure cards (wallet vs API key, persistent memory, portable reputation, hybrid agent-human DAOs); decentralized civilization simulator (3 modes × 4 levers × 6 outcomes scoring coordination, freedom, resilience, innovation, equity, civilization memory); knowledge cards including an honest-limit card; coordination-markets module honestly mapping yes/maybe/no/hybrid for what protocols can replace in firms; identity module with a dual-use risk card; philosophy module distinguishing four kinds of decentralization (geographic, economic, political, architectural); future-civilization module with autonomous research collectives, machine economies, programmable cities, and a how-this-could-go-wrong risk card; AI Civilization Theorist with five canned multi-paragraph responses (incl. self-pre-mortem) and free-text heuristic fallback; explicit guardrails section refusing token marketing, utopian determinism, buzzword padding, false novelty, and AI-risk hand-waving.

## Stack

Plain HTML, CSS, vanilla JavaScript. No build step. Bilingual via `[data-lang]` + CSS attribute selectors. Light/dark via `[data-theme]`. Syne display, Geist Sans body, Geist Mono mono. All diagrams inline SVG. JS uses template literals throughout — verified with `node --check`.
